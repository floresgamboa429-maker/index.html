# DataStock - Consultoría en Inventarios

Landing page de servicios de consultoría en inventarios, enfocada en:
- Optimización de procesos de almacén
- Implementación de métodos de valuación (Promedio, PEPS, UEPS)
- Creación de dashboards en Power BI y Google Sheets
- Cálculo de KPIs para áreas de almacén, análisis, contabilidad y ERP

## 🚀 Publicación
Este sitio está diseñado como proyecto estático (HTML, CSS, JS) y puede desplegarse fácilmente en plataformas como **Vercel** o **Netlify**.

## 📂 Estructura
- `index.html` → Página principal
- `consultores.html` → Sección de servicios
- `/assets` → Imágenes y estilos
